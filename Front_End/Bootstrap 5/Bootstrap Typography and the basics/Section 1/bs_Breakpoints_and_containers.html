2. Containers

The fundamental building block of Bootstrap

2.1 Containers types

  • .Container, which sets a max-width at each responsive breakpoint
  • .container-fluid, which is width: 100% at all breakpoints
  • .container-(breakpoints), which is width: 10% until the specified breakpoint
